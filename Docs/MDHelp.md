# Markdown 语法

## 1. 标题

用 `#` 表示标题，1-6 级。

```markdown
# 一级标题
## 二级标题
### 三级标题
```

注意 `#` 后要空格。

## 2. 段落与换行

段落之间用空行分隔。

```markdown
这是第一段。

这是第二段。MDHelp
```

段内换行用两个空格后回车，或 `<br />`。

```markdown
第一行  
第二行
```

## 3. 强调与删除线

```markdown
*斜体*  _斜体_
**加粗**  __加粗__
~~删除线~~
```

## 4. 列表

无序列表：

```markdown
- 项目 1
- 项目 2
```

有序列表：

```markdown
1. 第一项
2. 第二项
```

嵌套列表：

```markdown
- 水果
  - 苹果
  - 香蕉
- 蔬菜
  1. 西红柿
  2. 黄瓜
```

GitHub 任务列表：

```markdown
- [x] 已完成
- [ ] 未完成
```

## 5. 链接与图片

链接：

```markdown
[链接文字](https://example.com)
```

图片：

```markdown
![图片说明](https://via.placeholder.com/320x120 "图片标题")
```

参考式链接：

```markdown
这是一个[链接][1]

[1]: https://example.com
```

## 6. 代码

行内代码：

```markdown
这是 `const x = 1`。
```

代码块：

~~~markdown
```js
function hello() {
  return 'hi'
}
```
~~~

## 7. 引用块

```markdown
> 这是引用
> 这是同一个引用块
```

## 8. 表格

```markdown
| 列1 | 列2 | 列3 |
| --- | :--: | --: |
| 左对齐 | 居中 | 右对齐 |
```

## 9. 分隔线

```markdown
---
***
___
```

## 10. 转义字符

当你要显示 Markdown 符号时，用 `\`。

```markdown
\*不是斜体\*
```

## 11. 平台兼容

- GitHub 支持大部分语法，任务列表、表情、自动链接。
- VitePress 支持 Markdown 核心语法并可扩展公式、脚注、折叠。
- 普通 Markdown 渲染器只支持核心语法，请不要依赖平台扩展。

## 12. 常见问题

- 标题不生效：`#` 后需要空格。
- 换行不显示：用两个空格再回车。
- 表格错位：表头分隔行要 `---`，每列用 `|` 分隔。
- 图片不显示：检查路径或 URL 是否正确。
